# 🔍 Port Scanner

A simple **multithreaded TCP port scanner** built in Python to identify open ports on a target system.  
This project demonstrates basic network reconnaissance and Python scripting for cybersecurity.

---

## ⚙️ Features
- Scan **custom port ranges**  
- Support for **IP addresses and domain names**  
- Fast scanning using **multithreading**  
- Clear and readable output for open ports
